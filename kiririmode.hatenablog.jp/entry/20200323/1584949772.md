# vscode 上で Ricty のバッククォート(backtick)文字がおかしい

vscode で Ritcy フォントを指定してバッククォートを入力すると、以下のように後ろの文字が重なってしまうという問題があります。

例えば、以下の文字列を入力します。

```
`hoge`
`fuga`
```

vscode 上での表示はこちら。
