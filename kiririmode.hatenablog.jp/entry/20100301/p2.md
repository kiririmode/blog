EJB ではデプロイメント記述子が重要な役割を持つが、そのデプロイメント記述子は xml にて記述される。そして当然ながら、その xml は DTD に沿って記述されなければならない。
作成した xml が DTD に沿っているかを確認できるツールとして、xmllint がある。
